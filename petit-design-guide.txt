## 프로젝트 개요
- 치과 이름: 쁘띠치과 (Petit Dental Clinic), 대구
- GitHub Pages: https://petitfong.github.io/petitmagic/
- 모든 파일은 HTML 단일 파일로 제작
- 공통 CSS: petit-design.css (모든 HTML에 link 태그로 연결됨)

---

## 필수 규칙

### 1. CSS 파일 연결
모든 HTML 파일 <head>에 반드시 포함:
<link rel="stylesheet" href="petit-design.css">

### 2. 폰트
- 제목·강조: font-family: 'TmonMonsori', serif
  → h1, h2, hero 제목, 섹션 제목, 슬로건
- 본문·UI: font-family: 'Pretendard', sans-serif
  → body, nav, 버튼, 설명, 캡션

폰트 파일 위치: fonts/ 폴더 (GitHub repo 내)
- fonts/TmonMonsori_otf.otf
- fonts/TmonMonsori_ttf.ttf
- fonts/pretendard-variable.woff2

## 사용 금지 조합
- 딥그린(#007A54) 위에 검정 텍스트 → 가독성 없음
- 연그린(#E8F5F0) 위에 회색(#AAAAAA) 텍스트 → 대비 부족
- 딥그린(#007A54) 위에 라이트그린(#009B6B) 텍스트 → 대비 부족
- 카카오 노랑(#FEE500) 위에 흰색 텍스트 → 절대 금지
